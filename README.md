# BetterRecruiters 

The purpose of this project is to create a automated, AI-assisted, job finder for the purposes
of finding early career and low-skilled jobs

# Description

Designed using python, this program purposes is the provide the user with an AI-assisted pathway for
finding jobs in the current job market in the UK. It includes, but will not be limited to:

Job database - Locally stores a collection of jobs found through scrapping with API keys

ATS Integration - Seamless flow of information on job application, job status, job ratings, etc

AI Tailoring - Tailors both CV and cover letters to meet specific job keywords and needs 

Browser Automation - Implements onto an available browser to scrap through job listings  

Local LLM Support - Through the use of either Ollama or otherwisse, supports CV tailoring
and job hunting
